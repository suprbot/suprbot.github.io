Story about 7-DOF Manipulator - Part I. Hardware

2017-01-03 Technology

Robot Body (mechnical)

This blog will focus on the control system, so maybe we will talk this in detail latter.

The robot I uesed was build by ourselfs, but because of some reason, we will skip the details of this part. The only thing that we should attent is the arm structure. Because the complexity of 7-DOF kinematics, here we restrict the 7-DOF arm to S-R-S model without link offset, as shown in Fig. 1.

Fig. 1. 7-DOF manipulator model.

Control Systen

The control system can be divided into two parts, the main controller and the joint controller. A brief hardware layout of the control structure is shown in fig. 2. To the joint controller wo chose servo motor and servo driver with ethercat bus, we buy this directly form other vendors. And then to the main controller, the protagonist of this blog, we chose the polular signal-board computer, raspberry pi 2. we also can implent all of the things in a normal PC, and acturally the performance will be better.

Fig. 2. Brief hardware layout of the control structure.
